学习Linux基础命令第二天

Linux基础命令二

  • 查询历史命令记录：history

    历史执行记录保存在 ~/.bash_historty文件中

  • 用户和组：

    创建用户：useradd
    删除用户：userdel
    编辑用户：usermod
    创建组：groupadd
    删除组：groupdel
    编辑组：groupmod

  • 设置密码：passwd

  • 切换用户：su

  • 普通用户临时获取root权限：sudo
  • 筛选/过滤：grep

  • 管道/过滤：|

  • 退出：exit

  • 查看进程：ps

  • 杀死进程：kill

    【Linux基础命令】kill详解

  • 查看进程的端口号：netstat

拒绝错误 Connection refused：
1、查看服务器应用进程是否存在: ps -ef | grep xxxx
2、查看进程端口是否正确: netstat -nlp | grep $pid
3、查看端口允许访问的服务器地址段

  • 搜索：find
  • 安装软件包：yum

  • 查找命令的可执行文件位置：which

    说明：which命令使从用户的环境变量$PATH中逐层查找

  • 查找命令所在的相关路径：whereis

  • 帮助与详情：--help 和 man

    使用方法:
    ls –help #命令 –help
    man ls # man 命令
