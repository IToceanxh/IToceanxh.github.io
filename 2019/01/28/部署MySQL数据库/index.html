部署MySQL数据库的两种方式

一、使用yum方式安装数据库

二、 自定义安装

  1. 准备MySQL安装包，上传到linux

  2. 查询是否安装过MySQL

  3. 解压压缩文件

  4. 创建用户

    • 检查是否存在dba组和mysqladmin用户

    • 创建用户组

    • 创建用户

    • 复制配置文件到mysqladmin家目录

  5. 配置my.cnf配置文件

9.安装MySQL数据库

10.配置MySQL服务并引导自动启动

11.启动MySQL并查看进程和侦听
